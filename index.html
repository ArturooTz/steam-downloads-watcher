Leave your computer ON while your games are downloading.

Leave your computer ON while this application is runing and it will automatically turn off or hibernate when your downloads finish.

How does this work?

The Steam application uses a folder called 'downloads' to store current downloads and then moves the files to another folder. This program checks when the 'downloads' folder is empty and then triggers the computer shutdown/hibernation/sleep.

Feel free to check out the code in the GitHub repository to make comments and sugestions to help me to improve this little tool!
